# Pollution Tracker 🌫️

A small Python project that uses backtracking and wind data to estimate where pollution in a region is coming from.

## Files

- `download.py` – downloads weather / wind data  
- `main2.py` – runs the pollution backtracking simulation  

## About

This project tries to trace pollution backward instead of just assuming it was produced locally.

It uses:

- Python
- backtracking / particle simulation
- wind vector data from CDS API
